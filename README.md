# Typhoon
Transformer based neural network model for time series tasks.

## Structure

```
Typhoon/
    ├── LICENSE
    ├── README.md
    ├── __init__.py
    ├── model.py
    ├── modules.py
    └── utils/
        ├── __init__.py
        ├── functions.py
        └── trainer.py
```
* `utils/`
... This directory contain training script and sub functions.

## System Requirements
* OS: Unix-like
    * macOS Mojave 10.14.6
    * Debian 9.9
    
* Python 3.5 or later
* PyTorch 1.1.0
